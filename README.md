# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition data to identify factors affecting employee turnover.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI

## Key Insights
- Employees doing overtime are more likely to leave.
- Lower salary contributes to attrition.
- Some departments have higher turnover rates.

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Data Visualization
4. Business Insights
5. Dashboard Creation

## Dataset
IBM HR Analytics Employee Attrition Dataset
